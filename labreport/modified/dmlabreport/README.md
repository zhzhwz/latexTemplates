## `info`设置

使用 `\dmlabreportSetup` 命令。

- `courseName  = {<课程名称>},`，在花括号内填入课程名称；
- `teacherName = {<指导教师>},`，在花括号内填入指导教师；
- `labName     = {<实践名称>},`，在花括号内填入上机实践名称；
- `labNo       = {<实践作业编号>},`，在花括号内填入上机实践编号；
- `studentID   = {<学号>},`，在花括号内填入学号；
- `name        = {<姓名>},`，在花括号内填入姓名；
- `grade       = {<年级>},`，在花括号内填入年级；
- `date        = {<提交作业日期>},`，在花括号内填入日期。
